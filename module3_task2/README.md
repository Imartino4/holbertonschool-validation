Requirements to build website with GO-Hugo
======================

## Prerequisites:
    * Ubuntu 18.04
    * install git, go, HUGO, GNUmake

## Lifecycle:
    * build -> make build
    * post -> make POST_NAME=who-are-we POST_TITLE="Who are we" post
    * clean -> make clean
    * lint
    * package
    * unit-tests
    * integration-tests
    * validate
    * help -> make help
